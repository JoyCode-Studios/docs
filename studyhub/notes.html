Notes System

The Notes system in StudyHub is designed to be fast, simple, and fully offline. All notes are stored locally on your device using localStorage, meaning they remain available even without an internet connection.

How Notes Work

Each note you create is saved instantly. There is no save button â€” StudyHub automatically updates your stored notes whenever you type or edit content.

  • Notes are stored as plain text
  • Each note has a unique ID
  • All notes are saved in localStorage under a dedicated key
  • Notes persist until you manually delete them or clear browser data

Creating a Note

To create a new note, open the Notes page and click the New Note button. A blank editor will appear where you can begin typing immediately.

Editing Notes

Clicking any existing note will open it in the editor. Changes are saved automatically as you type, so you never need to worry about losing progress.

Deleting Notes

Each note includes a delete option. When you delete a note:

  • It is removed from the notes list
  • Its entry is removed from localStorage
  • The action cannot be undone

LocalStorage Key

Limitations

Because notes are stored locally:

  • They do not sync across devices
  • Clearing browser data will remove them
  • They are tied to the specific browser and device
