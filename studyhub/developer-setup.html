Developer Setup (React + Vite)

StudyHub Web is built using React, JavaScript, JSX, and Vite. This page explains how to clone the repository, install dependencies, and run the project locally. The setup is lightweight and fast, matching the simplicity of the StudyHub ecosystem.

Requirements

Before you begin, make sure you have the following installed:

  • Node.js (version 18 or later recommended)
  • npm or yarn
  • A modern code editor (VS Code recommended)

1. Clone the Repository

Open your terminal and run: 

git clone https://github.com/Skyline-Studies/studyhub-web.git
cd studyhub-web

2. Install Dependencies

Install all required packages using npm or yarn: 

npm install

or

yarn install

3. Start the Development Server

Vite provides a fast dev server with instant hotâ€‘reload. Start it using: 

npm run dev

or

yarn dev

The terminal will show a local development URL, usually: 

http://localhost:5173

4. Build for Production

To generate an optimised production build: 

npm run build

or

yarn build

The output will be placed in the dist/ folder.

5. Preview the Production Build

You can preview the built version locally using: 

npm run preview

or

yarn preview

6. Recommended Extensions (VS Code)

  • ESLint
  • Prettier
  • React Tools
  • Vite Helper

Once the dev server is running, you can begin editing components, pages, and logic. Vite will instantly reload changes, making development fast and smooth.
