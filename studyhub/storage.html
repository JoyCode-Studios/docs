What Happens If LocalStorage Is Cleared?

Clearing browser data will remove all StudyHub data stored locally. This includes:

  • all notes
  • all tasks
  • all subjects
  • your entire timetable

This action cannot be undone. StudyHub does not back up local data automatically.

Cloud Storage (Listify)

StudyHub does not store any cloud data itself. Instead, it integrates with Listify, which uses Cloud Firestore to store:

  • cloudâ€‘synced notes
  • lists
  • notes

Listify requires an account and handles authentication and syncing independently. StudyHub never stores your Listify login information.
