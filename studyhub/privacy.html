Privacy & Data Handling

StudyHub is designed with privacy, simplicity, and user control at its core. This page explains how StudyHub handles your data, what is stored locally, what is stored in the cloud (via Listify), and how your information is protected.

Local Data (Stored on Your Device)

StudyHub Web stores all core data using localStorage. This means:

  • your data never leaves your device
  • no servers are involved
  • no accounts are required
  • everything works offline

The following data is stored locally:

  • notes
  • tasks
  • subjects
  • timetable entries
  • theme preference (light/dark)

LocalStorage Keys

studyhub_notes
studyhub_tasks
studyhub_subjects
studyhub_timetable
studyhub_theme

These keys contain structured JSON data used by the app.

StudyHub Web does not collect:

  • email addresses

  • usage analytics

Everything stays on your device unless you explicitly use Listify.

Security Considerations

Because StudyHub uses localStorage:
