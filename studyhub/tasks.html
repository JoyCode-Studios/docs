Tasks System

The Tasks system in StudyHub is a simple but powerful to‚Äëdo manager designed to help you track homework, assignments, reminders, and daily study goals. All tasks are stored locally on your device using localStorage, making the system fast and fully offline.

How Tasks Are Stored

Every task you create is saved instantly. StudyHub uses a dedicated localStorage key to store all tasks as an array of objects. 

studyhub_tasks

Each task object contains:

  • The task ‚Äî the task name. However, this will save only locally, not to the cloud.
  • Listify tasks ‚Äî tasks in Listify, are not stored locally, but in the cloud. The following features in the list are specific to Listify only.
  • Completion status ‚Äî whether the task is completed or not. This is changed by clicking the checkbox next to the task.
  • Delete and modify buttons ‚Äî the ability to delete, and modify the task.

Adding a Task

To add a task, open the Tasks page and use the input field at the top. You can:

  • enter a task name

Once added, the task appears immediately in your task list.

Completing Tasks

Clicking the checkbox next to a task marks it as completed. Completed tasks:

  • move to the bottom of the list (depending on your UI layout)
  • remain stored in localStorage
  • can be un‚Äëcompleted at any time

Deleting Tasks

Each task includes a delete button. When you delete a task:

  • it is removed from the list
  • its entry is removed from localStorage
  • the action cannot be undone

Important Tasks

Tasks marked as important are highlighted and may appear in the dashboard‚Äôs ‚ÄúToday‚Äôs Tasks‚Äù section. This helps you focus on what matters most.

Due Dates

If you set a due date, StudyHub will:

  • display the date next to the task
  • highlight overdue tasks
  • show tasks due today on the dashboard

Offline Support

Because tasks are stored in localStorage, the entire system works offline. You can add, edit, complete, and delete tasks without an internet connection.

Limitations

The Tasks system is local‚Äëonly:

  • tasks do not sync across devices
  • clearing browser data will remove all tasks
  • tasks are tied to the specific browser and device
