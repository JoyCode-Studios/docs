Getting Started with StudyHub

StudyHub is a lightweight study companion built with React, Vite, and localStorage. It runs entirely in your browser and requires no account to begin. This guide will help you access StudyHub, understand the basic layout, and start using its core tools.

Accessing StudyHub

StudyHub Web is available on any modern browser. Simply visit the deployed URL (or run it locally if you're a developer). No installation or login is required.

  • Works on Chrome, Edge, Firefox, Safari
  • No extensions or plugins needed
  • All data is stored locally unless using Listify cloud sync

First-Time Setup

When you open StudyHub for the first time, your dashboard will be empty. StudyHub automatically creates storage areas in localStorage for:

  • notes
  • tasks
  • subjects
  • timetable entries

These will remain saved on your device until you clear your browser data.

Using StudyHub Offline

Because StudyHub stores everything in localStorage, it works fully offline. You can take notes, manage tasks, and update your timetable without an internet connection.
