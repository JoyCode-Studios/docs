Platform Support

StudyHub is designed to be accessible on a wide range of devices. The core version of StudyHub runs entirely in the browser, while a dedicated mobile app is planned for future release. This page explains which platforms are supported and how the experience differs across devices.

Web Platform (Primary)

The main version of StudyHub is built using React, JavaScript, JSX, and Vite. It runs directly in modern browsers without installation.

  • Google Chrome
  • Microsoft Edge
  • Mozilla Firefox
  • Safari
  • Opera

The web version is fully responsive and works on:

  • Windows
  • macOS
  • Linux
  • Chromebooks
  • iPadOS (Safari)
  • Android tablets (Chrome)

Mobile Web

StudyHub works on mobile browsers, but the layout is optimised primarily for desktop and tablet use. On smaller screens:

  • the sidebar collapses into a menu
  • the dashboard rearranges into a vertical layout
  • notes and tasks use a simplified editor

Mobile web is fully functional, but the upcoming mobile app will provide a smoother, more native experience.

Mobile App (Coming Soon)

A dedicated mobile version of StudyHub is planned using React Native. This will allow StudyHub to run as a native app on:

  • iOS
  • Android

Browser Requirements

StudyHub requires a browser that supports:

  • ES6 JavaScript
  • localStorage
  • CSS variables
  • Flexbox and Grid

These features are supported by all major browsers released after 2018. This is the list of supported browsers: Google Chrome latest, Microsoft Edge latest, Mozilla Firefox latest, Safari 12+, Opera latest and JoyCode Studios Nexion latest.
