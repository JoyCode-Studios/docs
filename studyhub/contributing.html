How to Contribute

1. Fork the Repository

Fork the repo you want to contribute to (web or mobile) and clone your fork locally.

2. Create a Feature Branch

git checkout -b feature/my-new-feature

3. Make Your Changes

Follow the existing code style and structure. Keep components modular and readable.

4. Test Thoroughly

Ensure your changes work across:

  • desktop browsers
  • mobile browsers
  • React Native simulators (for mobile)

5. Submit a Pull Request

Open a PR to the main branch of the appropriate repository. Include:

  • a clear description of your changes
  • screenshots if UIâ€‘related
  • any relevant issue numbers

Contribution Guidelines

  • Keep code clean and consistent
  • Use descriptive commit messages
  • Do not include unrelated changes in a single PR
  • Follow React and React Native best practices
  • Respect the licensing terms for each project
