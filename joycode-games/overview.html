Submitting Your Own Game

JoyCode Games welcomes contributions from developers of all skill levels. To submit a game, follow these guidelines:

  • Your game must run in a browser using HTML, CSS, and JavaScript.
  • All assets must be properly licensed (SIL OFL, MIT, CC0, etc.).
  • Place your game in the correct /genres/[category]/ folder.
  • Include an index.html file or a clearly named main file.
  • Optional: add a README.md explaining gameplay and controls.

Once added, your game can be linked from the main hub and showcased to players.
