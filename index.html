Welcome to JoyCode Documentation

This documentation system provides a unified, open‑source foundation for all JoyCode Studios projects. It includes a consistent layout, theming system, navigation structure, and reusable components designed to keep documentation clean, readable, and developer‑friendly.

What is this?

JoyCode Documentation is a universal docs template used across the JoyCode ecosystem. Whether you're documenting a language, runtime, library, or internal standard, this layout gives you a polished, professional starting point.

Features

  • Lightweight, dependency‑free layout
  • Dark mode support
  • Clean sidebar navigation
  • Readable typography
  • MIT‑licensed and fully open‑source

Get Started

Use the sidebar to explore the documentation sections. To create your own docs, copy this template into your project and customise it as needed.
